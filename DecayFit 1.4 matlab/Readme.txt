---------------------------------------------
--- Using the MATLAB version of DecayFit: ---
---------------------------------------------

To run DecayFit:

Method 1:
  1) Open MATLAB
  2) Make this folder your current directory (i.e. the folder containing DecayFit.m)
  3) In the command window type: DecayFit
  4) Press enter

Method 2:
  1) Double click DecayFit.m to open the file in the MATLAB Editor
  2) Press F5 to run the program (or press the green run-button in the Editor toolbar)


- Examples data is located in the data folder.
