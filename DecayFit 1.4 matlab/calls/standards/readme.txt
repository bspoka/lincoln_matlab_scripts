Standard functions are software-independent GUI functions used for most applications.

The GUI handles structure of the software must have the following fields:

 handles.name
 handles.version
 handles.splashScreenHandle
 handles.website
 handles.workdir
 handles.settings.startup.zoomMessage
 handles.settings.startup.checkforUpdates
 handles.settings.infobox.()

A default settings file must be located at workdir/calls/stateSettings/default.settings.
